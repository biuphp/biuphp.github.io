CURL简介

php的curl可以实现模拟http的各种请求，这也是php做网络爬虫的基础，也多用于接口api的调用。
PHP 支持 Daniel Stenberg 创建的 libcurl 库，能够连接通讯各种服务器、使用各种协议。libcurl 目前支持的协议有 http、https、ftp、gopher、telnet、dict、file、ldap。 libcurl 同时支持 HTTPS 证书、HTTP POST、HTTP PUT、 FTP 上传(也能通过 PHP 的 FTP 扩展完成)、HTTP 基于表单的上传、代理、cookies、用户名+密码的认证。

披星戴月，我两点一线的程序员生活。

发表于 2019-03-17 更新于 2019-11-18
本文字数： 1.4k 阅读时长 ≈ 1 分钟

今天这篇文章，没有技术性的东西，有的是对工作对生活的一点总结，也算是一种感悟吧。虽然我的草稿本上记载着两个月以来想分享给大家的技术点，但我却挤不出精力腾不出时间来好好梳理，以至于在那一刻想表达的内容，过了一段时间后，却没感觉的无奈。我一个东莞理工的大四学生，从12月份来开始来广州找工作，面试，到上班，已经有三个多月了。每天6点55分准时起床，7点14分坐上早班公交车，7点30到地铁站，8点出地铁，8点10分到公司，是的，基本上时间都是这样子的，每天在这条上班路上的人，我几乎都“认识”——每次在公交车上斗地主的中年男子，短头发背着小书包上学的女初中生，一股神似六神花露水一定不素颜的小姐姐，也许是缘分吧，我们都或在寒冷的冬晨、春分的朝阳下开启忙碌的一天。

与我一同实习的运营小姐姐总会是公司里的最早到的人儿，她是一个活泼开朗工作认真善于沟通的人，唯一的“缺点”是她睡得少（大概5/6小时就够了，她说的），这点让我很奇怪；好吧，8点20分吃完早餐，看了10分钟手机的公众号推送文章、刷刷新闻什么的，然后。。然后我就在工位上小趴一会，毕竟我可不像坐在我后面的运营小姐姐那样神奇，我是一个睡眠不够，无法工作的人，这也算是我给自己早起的一点点小奖励吧，可以这样懒懒的趴一下，你要知道我现在的公司是不要打卡上班，上班时间点靠自觉的，所以很多同事基本上9点快10点的样子到。（哈哈哈）
