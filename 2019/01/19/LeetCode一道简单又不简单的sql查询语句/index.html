知识点

CONCAT(str1,str2)

这里用到了mysql的concat函数，concat(str1,str2),连接参数产生的字符串。如有任何一个参数为NULL ，则返回值为 NULL
例如：

CONCAT_WS(separator,str1,str2)

Separator是分隔符，分隔符为 NULL，则结果为 NULL，但不会因为str1或者str2为null而结果值为null
