PHP7新特性

发表于 2019-05-21 更新于 2019-11-18
本文字数： 998 阅读时长 ≈ 1 分钟
  • 可为空（Nullable）类型

就如testeturn()函数一样，要么返回给定类型，要么返回null，其他则报错。

  • void函数

一个新的返回值类型void被引入。 返回值声明为 void 类型的方法要么干脆省去 return 语句，要么使用一个空的 return 语句。 对于 void 函数来说，NULL 不是一个合法的返回值。
