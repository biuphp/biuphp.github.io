php面试基础知识--字符串

发表于 2019-05-16 更新于 2019-11-18
本文字数： 1.7k 阅读时长 ≈ 2 分钟
  • strlen() : int

获取字符串长度

  • mb_strlen() : mixed

同样是获取字符串长度，但如果你要校验输入中文个数，就得使用此函数了。

  • explode() : array

函数返回由字符串组成的数组

  • trim ( string $str [, string $character_mask = “ \t\n\r\0\x0B” ] ) : string

去除字符串首尾处的空白字符（或者其他字符），字符如下：

“ “ (ASCII 32 (0x20))，普通空格符。
“\t” (ASCII 9 (0x09))，制表符。
“\n” (ASCII 10 (0x0A))，换行符。
“\r” (ASCII 13 (0x0D))，回车符。
“\0” (ASCII 0 (0x00))，空字节符。
“\x0B” (ASCII 11 (0x0B))，垂直制表符。

  • str_split() : string

将字符串转换为数组

  • 将驼峰转成下划线
